MuSimA

A Multi-Modal Simulator for ABAC Systems

Upload a ZIP file containing hand-drawn distribution images and a minimal JSON configuration.

JSON Schema / Format

Use this format for uploads or the editor. The example below is loaded from the server.

Required top-level fields
  • subject_size, object_size, environment_size (number)
  • rules_count (number)
  • subject_attributes_count, object_attributes_count, environment_attributes_count (number)
  • subject_attributes_values, object_attributes_values, environment_attributes_values (array of numbers)
  • subject_distributions, object_distributions, environment_distributions (array of objects)
Distributions format
Each item supports a distribution code like "N" (Normal), "P" (Poisson), "U" (Uniform). Some distributions may include parameters like mean/variance or lambda.
